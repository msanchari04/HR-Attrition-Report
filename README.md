 HR-Attrition-Report
Analysed HR dataset of 1,481 employee records across 38 attributes to identify key drivers of employee 
attrition. 
Built an interactive Power BI dashboard to analyse attrition by age group, gender, education field, job role, 
salary slab, and satisfaction metrics. 
Calculated attrition rate and key KPIs using DAX, enabling gender-wise and category-wise comparison of 
attrition trends. 
Derived actionable insights and retention recommendations by correlating attrition with salary levels, job 
satisfaction, and work environment factors.
